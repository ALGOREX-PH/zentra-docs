## In words

1. **Define & commit.** You author a private policy; `commitPolicy` registers its
   commitment and recipient root on-chain.
2. **Read state.** The SDK reads the agent's `AuthorityState` and derives the
   *effective prior* state for the current epoch.
3. **Prove.** snarkjs builds a Groth16/BN254 proof binding the action to the
   policy, recipient, amount, prior state, and a fresh nullifier.
4. **Verify.** The contract verifies the proof, checks the state binding, and
   confirms the nullifier is unused.
5. **Settle.** It writes the new state, marks the nullifier, and transfers funds
   from the agent to the recipient.
6. **Receipt.** It computes the Poseidon `action_id` and emits an `ActionReceipt`.
